Numpy datetime format

Published: Wed 23 August 2017
By frank

In Python.

tags: datetime Numpy Python

Python has a lot of ways to handle dates and times. This landscape of modules can be very messy for new users. Which format to use? What methods are suitable? It took me a while to figure out what ways were most convenient. To keep it simple I almost exclusively use the datetime module. And epoch timestamps or ISO strings for portability.

But today I needed to do calculations with dates in long Numpy vectors. So I wasn't looking forward to looping over each index and doing datetime stuff step by step. Most of the advice says to use Pandas for this, but I don't use it since I'm used to work with vectors and Numpy. Besides, it's slower than pure Numpy, and almost everything I do gets used in production so performance counts.

Fortunately Numpy has a datetime format as well, called datetime64. It's not as full fledged as the datetime module, but it worked really nice for my task. Here is how I converted my array with epoch timestamps to the Numpy datetime64 format.

>>> epochs = np.arange(1496567408, 1503416178)
>>> dt64 = np.array(epochs, dtype='datetime64[s]')
array(['2017-06-04T09:10:08', ..., '2017-08-22T15:36:17'])

You can change the unit with .astype and different letters in the bracket after the datetime64 designator. If you change the unit for example to days, all time information is truncated. This comes in handy when doing calculations. Suppose I have a giant list of epoch timestamps and I want to know which day of the month they are. By subtracting the year and month all that remains is the day, which I can easily cast as an integer. Plus 1 because days of the month don't use zero-based numbering.

>>> as_days = dt64.astype('datetime64[D]')
>>> as_months = dt64.astype('datetime64[M]')
>>> day_of_month = (as_days - as_months).astype(int) + 1
array([ 4, 4, 4, ..., 22, 22, 22])

A bit harder but still quite understandable is a way to get the day of the week of a long array with datetimes. Numpy has a function to determine whether a datetime is on a business day or not. You can supply a custom mask to select which days should return True or False. This mask has the form of a string with seven 1's or 0's. I used the bin function to create the string and .zfill to pad with zeros on the left side. The result is an array where Monday to Sunday is represented by 0 to 6.

>>> day_of_week = np.empty(np.size(dt64), dtype=int)
>>> for i in range(7):
>>>     weekmask = bin(1 << (-i + 6))[2:].zfill(7)
>>>     bools = np.is_busday(dt64.astype('datetime64[D]'), weekmask)
>>>     day_of_week[bools] = i
# 4-6 was a Sunday, 22-8 a Tuesday
array([6, 6, 6, ..., 1, 1, 1])

By using these methods and some more Numpy magic I was able to remove Pandas from a piece of code, which led to a 7-fold increase in speed of the entire module. Not bad!
