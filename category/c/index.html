Simple C code to show size of types

Published: Tue 17 October 2017
By frank

In C.

Learning to code in C after working in Python for a while means slowly uncovering the intricate mechanics underlying the smooth Python experience. One of my first attempts to write a simple script to collect sensor data promptly crashed, and unlike Python you don't get exception messages. I was able to find out it had returned exit code 139, but that didn't really help much. I quickly learned not to rely on convenient exception messages like in Python, but to really understand the variables and control flows I was calling into existence.

One of the first scripts I used was to show how the variable types in C differ on different systems. It's a good reminder of the basic level of C and handy to have lying around. It learned me that on a Raspberry Pi an int and a long have the same size.

#include <stdio.h>
#include <limits.h>
#include <float.h>

int main() {
    printf("size of char: %d \\n", sizeof(char));
    printf("number of bits in char: %d \\n", CHAR_BIT);
    printf("min value of regular char: %d \\n", CHAR_MIN);
    printf("max value of regular char: %d \\n", CHAR_MAX);
    printf("size of int: %d \\n", sizeof(int));
    printf("min value of int: %d \\n", INT_MIN);
    printf("max value of int: %d \\n", INT_MAX);
    printf("max value of unsigned int: %u \\n", UINT_MAX);
    printf("size of long: %d \\n", sizeof(long));
    printf("min value of long: %d \\n", LONG_MIN);
    printf("max value of long: %d \\n", LONG_MAX);
    printf("max value of unsigned long: %u \\n", ULONG_MAX);
    printf("size of long long: %d \\n", sizeof(long long));
    printf("min value of long long: %d \\n", LLONG_MIN);
    printf("max value of long long: %d \\n", LLONG_MAX);
    printf("max value of unsigned long long: %u \\n", ULLONG_MAX);
    printf("size of float: %d \\n", sizeof(float));
    printf("The maximum value of float = %.10e\\n", FLT_MAX);
    printf("The minimum value of float = %.10e\\n", FLT_MIN);
    printf("The number of digits in the number = %.10e\\n", FLT_MANT_DIG);
    printf("size of double: %d \\n", sizeof(double));
    printf("The minimum value of double = %.10e\\n", DBL_MIN);
    printf("The maximum value of double = %.10e\\n", DBL_MAX);
    printf("The number of digits in the number = %.10e\\n", DBL_MANT_DIG);
    return 0;
}
